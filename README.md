QA Test Case Manager

Simple React application for managing manual QA test cases.

Features
- Add / edit / delete test cases
- Status management (Passed, Failed, Blocked)
- Search and filter
- Dashboard with statistics
- Data persistence with localStorage

Tech Stack
- React
- Vite
- Context API
- Recharts
